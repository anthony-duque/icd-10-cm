Load the ICD code file to the database
 
  1. Download the zip file from website: https://www.cms.gov/medicare/coding/icd10
  2. Extract the zip file to a desired folder
  3. The text file: cdxxcm-codes-yyyy.txt contains the CM codes
  4. The pdf file: icdxxOrderFilesyyyy.pdf contains the record structure of the text file
